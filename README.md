# 狙击步枪俱乐部 🎯

一个恶搞性质的静态网页项目，纯粹为了好玩而制作！

## 项目介绍

⚠️ **请注意：这是一个恶搞网页，仅用于娱乐目的！**

本项目是一个简单的静态网页，展示了两款经典的狙击步枪：
- **M24 狙击步枪** - 美国雷明顿公司生产的高精度狙击武器
- **Kar98K 毛瑟步枪** - 二战时期德国军队的标志性步枪

## 功能特性

- 🎯 展示经典狙击步枪图片
- 🔗 点击图片可跳转到百度百科了解详细信息
- 📱 响应式设计，支持多种设备浏览

## 技术栈

- HTML5
- CSS3
