# 狙击步枪俱乐部 🎯

一个恶搞性质的静态网页项目，纯粹为了好玩而制作！

## 项目介绍

⚠️ **请注意：这是一个恶搞网页，仅用于娱乐目的！**

本项目是一个简单的静态网页，展示了四款经典武器：
- **M24 狙击步枪** - 美国雷明顿公司生产的高精度狙击武器
- **Kar98K 毛瑟步枪** - 二战时期德国军队的标志性步枪
- **PP-19 野牛** - 俄罗斯生产的冲锋枪，以其独特的螺旋弹匣设计闻名
- **内格夫轻机枪** - 以色列国防军制式轻机枪

## 功能特性

- 🎯 展示经典武器图片
- 🔗 点击图片可跳转到百度百科了解详细信息
- 📱 响应式设计，支持多种设备浏览

## 技术栈

- HTML5
- CSS3
