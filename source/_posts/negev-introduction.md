## 内格夫轻机枪简介

内格夫（NEGEV）是以色列军事工业（IMI）研制的5.56mm轻机枪，以巴勒斯坦南部地区命名，现为以色列国防军的制式班用机枪。该枪设计灵感源自FN Minimi，但在沙漠环境中的可靠性更胜一筹。

### 基本信息

| 项目 | 详情 |
|------|------|
| 类型 | 轻机枪 |
| 研制国家 | 以色列 |
| 研制年份 | 1985-1995年 |
| 口径 | 5.56×45mm NATO |
| 弹匣/弹链容量 | 35发弹匣 / 150-200发弹链 |
| 有效射程 | 1000米 |
| 射速 | 650-850发/分钟 |
| 空枪重量 | 7.6公斤 |

### 发展历程

```text
1985年 - 以色列国防军提出需求
1993年 - 早期原型交付精锐部队测试
1995年 - 最终定型
1997年 - 正式装备以色列国防军
```

### 设计特点

1. **双供弹系统** - 可使用150/200发金属弹链或标准35发弹匣（加利尔/M16弹匣），战场适应性极强
2. **快速更换枪管** - 提供标准长枪管和短管突击型两种选择，可根据任务需求快速切换
3. **三档气体调节器** - 位置1用于弹匣供弹，位置2用于正常弹链供弹，位置3用于恶劣环境下增大导气量
4. **折叠枪托** - 枪托可向右折叠，便于在车辆和狭窄空间中携带
5. **皮卡汀尼导轨** - 可安装各类光学瞄具、激光指示器等战术附件

### 性能参数

- **初速**：950米/秒
- **理论射速**：650-850发/分钟（可调）
- **枪管长度**：460毫米（标准型）/ 330毫米（突击型）
- **全枪长**：1020毫米（枪托展开）/ 780毫米（枪托折叠）
- **供弹方式**：M27弹链 / Galil弹匣 / STANAG弹匣

### 实战应用

内格夫在以色列国防军的多次实战中表现出色，包括2006年以黎冲突、加沙战争等。以色列士兵发现，内格夫在沙漠环境中比FN Minimi更可靠，且折叠枪托设计便于在装甲车辆中携带。该枪也被出口到多个国家的特种部队使用。

### 游戏中的内格夫

在CS:GO等射击游戏中，内格夫以150发超大弹匣和极高射速著称。虽然价格不菲且弹道扩散较大，但持续火力压制能力无人能敌。前10发子弹弹道不稳定，但持续射击后弹道会高度集中于一点，适合防守型压制作战。
