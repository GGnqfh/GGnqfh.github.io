## PP-19野牛冲锋枪简介

PP-19"野牛"（俄语：ПП-19 Бизон）是俄罗斯伊孜玛什公司于1993年研制的一款冲锋枪，由AK之父卡拉什尼科夫之子维克多·卡拉什尼科夫与SVD设计者之子阿列克谢·德拉贡诺夫联合设计，堪称枪坛佳话。

### 基本信息

| 项目 | 详情 |
|------|------|
| 类型 | 冲锋枪 |
| 研制国家 | 俄罗斯 |
| 研制年份 | 1993年 |
| 口径 | 9×18mm马卡洛夫 |
| 弹匣容量 | 64发（螺旋弹筒） |
| 有效射程 | 200米 |
| 射速 | 650-700发/分钟 |
| 空枪重量 | 2.1公斤 |

### 发展历程

```text
1993年 - 伊孜玛什公司开始研制
1995年 - 完成设计定型
1996年 - 正式列装俄罗斯内务部
```

### 设计特点

1. **螺旋弹筒供弹** - 采用独特的64发圆筒形螺旋弹筒，纵向安装在枪管下方，可兼做护木，大幅提升了持续火力
2. **AK血统** - 60%的零部件与AK-100系列通用，机匣、复进簧、折叠枪托等均沿用AK步枪成熟结构
3. **自由枪机原理** - 舍弃AK的导气式设计，采用最简单的自由枪机原理，闭膛待击，提高了射击精度
4. **轻便紧凑** - 空枪仅重2.1公斤，枪托折叠后更加紧凑，非常适合室内近战（CQB）环境

### 性能参数

- **初速**：约315米/秒
- **理论射速**：650-700发/分钟
- **枪托展开全长**：660毫米
- **枪托折叠全长**：425毫米
- **供弹方式**：64发螺旋弹筒

### 实战应用

PP-19"野牛"自1996年列装以来，经受了严苛的实战检验。在北高加索地区的反恐行动中，其64发超大容量弹匣在城市巷战中展现出持续压制火力的明显优势，深受俄罗斯"阿尔法"突击队和"雪豹"特种部队的青睐。

### 游戏中的野牛

在CS:GO等射击游戏中，PP-19野牛凭借64发超大弹匣和较低的价格，成为经济局的利器。虽然单发伤害不高，但超大的弹容量和较快的换弹速度使其在压制和清场方面表现出色，尤其适合非军事区的压制作战。
